Server FAQs.

Here are some frequently asked questions that we get and the answers to most of them.

Does the server have seasonal resets and if so how often do we reset?
We do not have a exact estimate of when the server resets, but we will announce in the discord server before hand when it resets so you know when it does.
Is the server free to play?
Yes the server is free to play, though there are some advantages in the server Store that helps with progressing in the server, But most stuff you get from the server store you can get without spending money.
Does the server have active moderators?
Though we do not have many Moderators/Admins, we are currently looking for and working on a Moderation/Administration team and building up our staff.
Is the owner active and does he communicate and interact with players?
The owner is absolutely active and is always interacting with players and hosting events with the server, If you ever have question, dm a admin or the owner and they will respond to you.
